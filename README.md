# **eos.sh**

*A small ritual for new machines.*

This script installs a focused set of developer tools, applications, and shell configuration so new machines can be set up quickly and consistently.

Named after **Ἠώς (Eos)**, the goddess of dawn.

---

## **What it does**

- Installs Homebrew
- Sets Fish as the default shell
- Clones dotfiles using `yadm`
- Installs developer tools and CLI utilities
- Installs `nvm` and Node.js LTS
- Installs Fisher and Fish plugins
- Installs a small set of desktop applications

---

## **What gets installed**

### Applications

- Zed  
- Ghostty  
- Google Chrome  
- Figma  
- Raycast  
- Discord  
- 1Password  

### Shell

- Fish shell  
- Fisher plugin manager  
- Plugins:
  - `nvm.fish`
  - `fzf.fish`
  - `z`

### Developer tools

- `git`, `gh`
- `nvm` + Node.js LTS
- `fzf`
- `z`
- `ripgrep`, `fd`
- `bat`, `eza`, `tree`
- `jq`, `wget`, `curl`, `htop`, `tldr`
- `yadm`

### System

- Homebrew  
- Xcode Command Line Tools
